# make-color-accessible 

produce colors that meet web content accessibility guidelines

## Installation

```sh
npm install make-color-accessible --save
```


## Tests

```sh
npm install
npm test
```

## Dependencies

- [color2](https://github.com/dfcreative/color): Stateful implementation of color
